Project 1: Random Number Guessing Game

Algorithm:
1.Generate a random number within a specified range.

2.Ask the player to select a difficulty level, which determines the number of attempts.

Loop until the player guesses correctly or runs out of attempts:

Prompt the player to enter a guess.

3.Check if the guess matches the target number.

If correct → game ends with a success message.

If too high → give a hint to guess lower.

If too low → give a hint to guess higher.

Decrement remaining attempts.

4.If all attempts are used without guessing correctly → display a failure message.
